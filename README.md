# oomck-bot

## Getting started

### Dependancies

- [Docker] (https://www.docker.com/get-started)
- [Python 3] (https://www.python.org/downloads/)

### Setup Environment

1. Start ElasticSearch container `docker run -d --name elasticsearch -p 9300:9300 -p 9200:9200 elasticsearch:7.6.2`

2. `python init.py`
